📊 Sales Performance Dashboard (Power BI)
🚀 Overview

This project is an interactive Sales Performance Dashboard built using Power BI to analyze business sales data and generate actionable insights.

📌 Objectives
Analyze overall sales performance
Track key business metrics (Revenue, Profit, Profit Margin)
Identify top-performing products
Understand regional sales distribution
Monitor monthly trends
📊 Key Features
✅ KPI Cards (Total Revenue, Total Profit, Profit Margin)
📈 Monthly Revenue Trend Analysis
🏆 Top Products by Revenue
🌍 Revenue Distribution by Region
🔄 Revenue vs Profit Comparison
🎯 Interactive Filters (Region & Product)
📊 Key Insight

👉 Revenue dropped in February but showed a strong recovery in March, indicating possible seasonal or operational factors.

🛠 Tools & Technologies
Power BI
DAX (Data Analysis Expressions)
Excel / CSV Dataset
